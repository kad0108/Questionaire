# Questionaire

> 基于 VueJs + NodeJS + MongoDB 的多人管理问卷平台

### 前端：

```
http://localhost:8888/#/
```

* Vue
* Vue-Cli
* Vue-Resource
* Vue-Router
* SessionStorage

### 后端：

监听8088端口

* Node
* Express
* MongoDB(Mongoose)
* JWT

### 跨域

Vue中通过[proxyTable](https://vuejs-templates.github.io/webpack/proxy.html)来开启一个代理服务跨域请求后端API
